# 3D Graphics Demo

The 3D Graphics demo showcases interactive 3D rendering capabilities using Three.js.

## Features

- Interactive 3D teapot model
- Smooth rotation animation
- Orbit controls for camera manipulation
- Professional lighting and materials

## How to Use

1. Select "3D Graphics" from the demo list in the left sidebar
2. The 3D teapot will appear and automatically rotate
3. Use your mouse to interact with the scene:
   - **Left click + drag**: Rotate the camera around the model
   - **Right click + drag**: Pan the camera
   - **Scroll wheel**: Zoom in and out
   - **Middle click + drag**: Pan the camera (alternative)

## Technical Details

The demo uses Three.js, a popular WebGL library, to render 3D graphics directly in the browser. The teapot is constructed from basic geometric primitives including spheres, cones, cylinders, and torus shapes.

The scene includes:
- Ambient lighting for overall illumination
- Directional lighting for depth and shadows
- Phong material for realistic surface reflections
- Orbit controls for intuitive camera manipulation
